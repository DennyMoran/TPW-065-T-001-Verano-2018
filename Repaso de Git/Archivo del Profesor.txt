Pasos para trabajar con git:

1ro: git pull <nombre del repositorio>
2do: desde la terminal (consola) entramos en el directorio del repositorio.
3ro: Realizar los cambios
4to: En la terminal: git status -- Esto me deja saber los archivos que se han modificado.
Archivo rojo === NO LISTO PARA COMMIT
Archivo verde === LISTO PARA COMMIT.
5to: git add .
6to: git commit -m " SU mensaje "
7mo: git push origin master
